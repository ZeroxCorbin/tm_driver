# tm_driver

This is a set of classes to communicate to the Omron\Techman cobots.

It supports Ethernet Slave and Listen Node functionality.

It is a supset of code from the repository;
https://github.com/TechmanRobotInc/tmr_ros2
All ROS references have been removed.


INSTRUCTIONS;

Visual Studio: Use the "Open Folder" File command to open the root directory of this repo.
VSCode: Open the root folder of the repo.
Linux: Run CMake, then make.
